📜 A* Algorithm Explained

🧠 What is A* Algorithm?

A* (A-Star) is a powerful pathfinding algorithm used to find the shortest and most efficient path between two points.

It works using:

f(n) = g(n) + h(n)

g(n) → Cost from start
h(n) → Estimated distance to goal
f(n) → Total cost

⚙️ Step-by-Step Procedure

STEP 01

Initialize

Start node is added to the open list with initial cost.

STEP 02

Explore

Neighbors are checked and evaluated using cost function.

STEP 03

Select Best

The node with lowest f(n) is chosen for next step.

STEP 04

Repeat

This continues until the goal node is reached.
